### Описание проекта
Islam Tour — это учебный веб-проект туристического сайта.  
Сайт предназначен для просмотра туристических направлений, ознакомления с турами и получения основной информации о путешествиях.

### Цель проекта
Создать простой и понятный туристический сайт с удобной навигацией и современным интерфейсом.

### Функционал
- Просмотр списка туров
- Описание направлений
- Главная страница с общей информацией
- Адаптивная верстка
- Простая навигация по сайту
