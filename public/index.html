О проекте

Учебный проект для Ислам Шарапов (АТУ). Тема: разработка веб-приложения туристического агентства с элементами ИИ для туристической отрасли (направления: Египет, Дубай).

Что реализовано
  • Витрина туров + фильтр
  • Рекомендации по бюджету/стилю/дням
  • AI-чатбот через OpenAI API
  • Монолит: один сервер
Можно улучшить
  • Заявка на бронирование + сохранение
  • История чата на сервере
  • Стриминг ответа (эффект “печатает…”)
